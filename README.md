# Radio Interview Question Generator
A lightweight, browser-based radio interview question generator for hosts, producers, and artist prep.

## Features
- Category-based question generation
- One-click regenerate
- Copy to clipboard
- Hosted via GitHub Pages

## Setup
1. Clone or fork the repo
2. Enable GitHub Pages (Settings → Pages → main branch)
3. Open the live URL

## Customization
- Add questions in `script.js`
- Adjust styling in `style.css`
- No backend required
